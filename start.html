About

This is a python module that makes it easy to implement mod features for Counter-Strike: Global Offensive. This module works by reading and writing memory to the process. To visit the Pypi page click here.

Installation

Step One

Install the pip package manager.

Instructions on installing pip are available here

Step Two

Installing CSGOMemPy with Pip.

Installing with pip package manager:

pip install CSGOMemPy

or try:

python -m pip install CSGOMemPy

Step Three

Importing CSGOMemPy into your Python project

Example
from CSGOMemPy import *

Documentation

Glow
GlowAll( dll, R, G, B, R, G, B) 
from CSGOMemPy import *

def main():
    glowAll("client.dll", 1, 0, 0, 0, 1, 0)


if __name__ == '__main__':
   main()

In our glowAll() line we are inputing our client dll string and then 6 RGB (Red, Green, Blue) float values. The first 3 are for the terrorist team and the second 3 are for the counter-terrorist team. For example, in the code above we are setting the terrorist glow to be red and the counter-terrorist glow to be green.

GlowT( dll, R, G, B ) 
from CSGOMemPy import *

def main():
    glowT("client.dll", 1, 0, 0)


if __name__ == '__main__':
   main()

In our glowT() line we are inputing our client dll string and then 3 RGB float values.These values represent the Red, Green and Blue channels of the terrorist team's glow color.

GlowCT( dll, R, G, B ) 
from CSGOMemPy import *

def main():
    glowCT("client.dll", 0, 1, 0)


if __name__ == '__main__':
   main()

In our glowCT() line we are inputing our client dll string and then 3 RGB float values.These values represent the Red, Green and Blue channels of the counter-terrorist team's glow color.

Triggerbot

triggerbot( dll )

from CSGOMemPy import *

def main():
    triggerbot("client.dll")


if __name__ == '__main__':
   main()

For the triggerbot function, all we need to do is input the client dll as a string, then it will automatically shoot upon looking at an enemy

Recoil-Control System

rcs( dll )

from CSGOMemPy import *

def main():
    rcs("client.dll")


  if __name__ == '__main__':
     main()

In the rcs function, we need to input our client dll as a string, then there will be no recoil on your guns.
